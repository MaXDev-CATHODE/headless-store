# ⚡ Headless Lightning Store (Portfolio Demo)

[![Lighthouse 100](https://img.shields.io/badge/Lighthouse-100-brightgreen.svg)](https://maxdev-cathode.github.io/headless-store/)
[![Astro](https://img.shields.io/badge/Astro-4.0-blue.svg)](https://astro.build/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-v4.0-38B2AC.svg)](https://tailwindcss.com/)

Ekstremalnie szybkie demo sklepu internetowego zbudowane w architekturze **Headless** przy użyciu **Astro**. Projekt stworzony pod uwiarygodnienie skilli e-commerce w portfolio, skupiony na perfekcyjnej wydajności (Core Web Vitals) i nowoczesnym designie.

> **Zero JavaScriptu na kliencie** (tam gdzie to możliwe). **Lighthouse 100/100**. **Błyskawiczne ładowanie.**

## 🚀 Kluczowe Funkcje
- **Performance-First:** Wynik 100/100 w Google PageSpeed Insights.
- **Modern Stack:** Astro (SSG), Tailwind CSS v4, TypeScript.
- **Glassmorphism UI:** Nowoczesna, ciemna estetyka z efektem szkła i płynnymi animacjami.
- **Dynamic Routes:** Automatycznie generowane karty produktów z lokalnych danych JSON.
- **Responsive Design:** Perfekcyjne wyświetlanie na każdym urządzeniu (Mobile First).

## 📦 Jak to działa?
1. **SSG (Static Site Generation):** Każda strona jest pre-renderowana do czystego HTML/CSS podczas budowania.
2. **Asset Optimization:** Obrazy są automatycznie konwertowane do formatów WebP/AVIF i skalowane.
3. **Zero JS Runtime:** Astro eliminuje zbędny JavaScript, wysyłając tylko to, co niezbędne do interakcji.

## 🛠 Instalacja Lokalna
```bash
git clone https://github.com/MaXDev-CATHODE/headless-store.git
cd headless-store
npm install
npm run dev
```
